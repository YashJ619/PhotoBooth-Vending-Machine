********************************************INITIALIZATION**************************************************************
IF YOU CHANGE PC OR FORMAT THIS PC THAN FOLLOW THESE STEPS
1. INSTALL PYTHON "https://www.python.org/downloads/".
2. RUN CMD IN ADMIN MODE.
3. TYPE "pip install pyserial".
4. TYPE "pip instal watchdog".
5. INSTALL ARDUINO IDE "https://www.arduino.cc/en/software".

**********************************************MANUAL TEST***************************************************************

TO MANUALY CHECK ALL RAKES
1. OPEN ARDUINO IDE.
2. OPEN SERIAL MONITOR.
3. SET BAUDRATE TO 115200.
4. FOR 1st RAKE TYPE "rake1on".
5. FOR 2nd RAKE TYPE "rake2on".
6. FOR 3rd RAKE TYPE "rake3on".
7. FOR 4th RAKE TYPE "rake4on".
8. FOR 5th RAKE TYPE "rake5on".

************************************************SETUP*******************************************************************

FOR SETUP
1. RUN main.py.
2. CLICK ON RAKE BUTTON ON SCREEN. [ENABLE = GREEN / DISABLE = RED].
3. TYPE NUMBER OF OBJECTS. NOTE: MAX 10.
4. ENTER PORT NO. WHERE ARDUINO CONNECTED.
5. ENTER DIRECTORY PATH YOU WANT TO MONITOR.
6. CLICK ON APPLY. NOTE: IF EVERY THING IS CORRECT IT WILL SHOW "OK" ELSE "ERROR".
7. COLSE THE WINDOW. NOTE: DONT COLSE THE COMMAND WINDOW OF PYTHON THIS WILL END THE PROGRAM.
